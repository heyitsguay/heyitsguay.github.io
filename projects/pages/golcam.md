# golcam

<video width="100%" autoplay muted loop poster="./media/golcam/poster1.png">
		<source src="./media/golcam/clip1.mp4" type="video/mp4">
</video>

Born out of an artech hackathon, **golcam** pulls silhouettes from a webcam and uses them to alter the rules obeyed by [Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)-like (GoL) cellular automata.

<video width="100%" autoplay muted loop poster="./media/golcam/poster2.png">
		<source src="./media/golcam/clip2.mp4" type="video/mp4">
</video>

The code allows a user to set rules in different parts of the screen based on silhouette brightness, then watch patterns grow and change in real time along with their motion.
